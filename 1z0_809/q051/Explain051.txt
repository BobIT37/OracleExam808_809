- Stream elements Type: Int

- Return Type: R

UnaryOperator<T>
- Argument Type: T
- Return Type: T

IntUnaryOperator

- Second Argument Type: U

map(IntUnaryOperator mapper)

- จากโจทย์ newStream.forEach(System.out::print); จะเริ่มวนลูปแล้วเอาค่า 1, 2, 3 ไปใส่ใน y และทำการปริ้นทีละตัว
