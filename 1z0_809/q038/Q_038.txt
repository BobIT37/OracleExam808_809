Which statement is true about the single abstract method of the java.util.function.Function
interface?

Question No : 38


	A. It accepts one argument and returns void.
	B. It accepts one argument and returns boolean.
	C. It accepts one argument and always produces a result of the same type as the
	argument.
	D. It accepts an argument and produces a result of any data type.


======================================

Answer: D
