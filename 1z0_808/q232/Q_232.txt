Question No : 232
Given the code fragment:

4.public static void main (String[] args){
5.		boolean opt = true; 
6.		switch(opt) { 
7.			case true :
8. 				System.out.print("True"); 
9.				break; 
10.			default: 
11.				System.out.print("***");
12.		} 
13.		System.out.println("Done");
14.}

Which modification enables the code fragment to print TrueDone?
A. 	Replace line 5 With String result = "true";
	Replace line 7 with case "true":
B. 	Replace line 5 with boolean opt = l;
	Replace line 7 with case 1=
C. At line 9, remove the break statement.
D. Remove the default section.

Answer: A

Explanation: จากโจทย์ถามว่าจะต้องแก้โค้ดส่วนไหนจะทำให้ปลิ้น TrueDone ออกมา
A. ถูก เพราะ แทนบรรทัด 5 ด้วย String result = "true"; และ แทนบรรทัด 7 ด้วย case "true": จะทำให้ปลิ้น TrueDone ซึ่งตรงกับที่โจทย์ต้องการ
B. ผิด เพราะ แทนบรรทัด 5 ด้วย boolean opt = l; จะ compile ไม่ผ่านเพราะ  boolean opt รับค่า int ไม่ได้ และ แทนบรรทัด 7 ด้วย case 1= จะ compile ไม่ผ่านเพราะใส่ = ไม่ได้ต้องใส่ :
C. ผิด เพราะ ที่บรรทัด 9 ลบ break ออก  compile ไม่ผ่านอยู่ดีเพราะ switch รับ data type ที่เป็น boolean ไม่ได้
D. ผิด เพราะ ลบเงื่อนไข default: ออก  compile ไม่ผ่านอยู่ดีเพราะ switch รับ data type ที่เป็น boolean ไม่ได้
